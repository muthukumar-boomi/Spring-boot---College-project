# SS-College Management System

A web-based college management system developed with **Spring Boot** and **Thymeleaf**. It streamlines administrative tasks such as student admission, academic records, attendance, fees, exams, and staff management.

## 🚀 Features
- **User Authentication** (admin / staff / student)
- **Student Admission** – add, edit, view, list
- **Academic Records** – manage marks, grades, and progress
- **Attendance Management** – daily tracking and reports
- **Fee Management** – collect and track fees, generate receipts
- **Exam Results** – publish and view results
- **Staff Records** – manage faculty details
- **Timetable** – schedule management
- **Responsive UI** using Bootstrap (in templates)

## 🛠️ Technologies
- Java 21
- Spring Boot 3.x
- Spring Data JPA
- Thymeleaf (templating)
- MySQL / PostgreSQL (database)
- Maven (build tool)
- Bootstrap (frontend styling)
